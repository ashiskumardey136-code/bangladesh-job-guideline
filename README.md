# চাকরি প্রস্তুতি — BCS ও সরকারি চাকরির প্রশ্নব্যাংক অ্যাপ

এটি একটি সম্পূর্ণ Flutter প্রজেক্ট। প্রফেশনাল UI, ক্যাটাগরি-ভিত্তিক কুইজ, স্কোর সিস্টেম এবং
AdMob বিজ্ঞাপন বসানোর কাঠামো প্রস্তুত করা আছে।

## যা যা আছে
- `assets/question_bank.json` — সব প্রশ্ন এখানে সংরক্ষিত (নতুন প্রশ্ন যোগ করতে শুধু এই ফাইল এডিট করুন)
- `lib/main.dart` — অ্যাপের থিম ও এন্ট্রি পয়েন্ট
- `lib/screens/home_screen.dart` — ক্যাটাগরি সিলেকশন স্ক্রিন + ব্যানার অ্যাড
- `lib/screens/quiz_screen.dart` — কুইজ চালানোর স্ক্রিন
- `lib/screens/result_screen.dart` — ফলাফল স্ক্রিন + ইন্টারস্টিশিয়াল অ্যাড
- `lib/data/ad_helper.dart` — AdMob কনফিগারেশন (⚠️ এখন টেস্ট আইডি বসানো আছে)

## ধাপ ১: স্থানীয়ভাবে চালানো
```bash
flutter pub get
flutter run
```
(আপনার কম্পিউটারে Flutter SDK ইনস্টল থাকতে হবে — flutter.dev থেকে ইনস্টল করুন)

## ধাপ ২: প্রশ্ন সংখ্যা বাড়ানো
`assets/question_bank.json` ফাইলের প্রতিটি ক্যাটাগরির `questions` লিস্টে নিচের ফরম্যাটে
নতুন প্রশ্ন যোগ করুন:
```json
{"q": "প্রশ্নের লেখা?", "options": ["ক", "খ", "গ", "ঘ"], "answer": 1}
```
`answer` হলো সঠিক অপশনের index (০ থেকে শুরু)। নতুন ক্যাটাগরি চাইলে `categories` অ্যারেতে
নতুন অবজেক্ট যোগ করুন এবং `home_screen.dart`-এর `_icons` ম্যাপে একটা আইকন যোগ করুন।

## ধাপ ৩: আসল AdMob আইডি বসানো (পাবলিশের আগে অবশ্যই করতে হবে)
1. https://admob.google.com এ অ্যাকাউন্ট খুলুন
2. একটি অ্যাপ তৈরি করে Banner ও Interstitial ইউনিট বানান
3. `lib/data/ad_helper.dart` ফাইলে টেস্ট আইডিগুলো আপনার আসল আইডি দিয়ে বদলে দিন
4. Android-এর জন্য `android/app/src/main/AndroidManifest.xml`-এ আপনার AdMob App ID
   `<meta-data android:name="com.google.android.gms.ads.APPLICATION_ID" android:value="ca-app-pub-xxxxxxxx~xxxxxxxx"/>`
   হিসেবে যোগ করতে হবে (Flutter নতুন প্রজেক্ট বানালে এই ফোল্ডার অটো তৈরি হবে)

## ধাপ ৪: APK/App Bundle বানানো ও পাবলিশ করা
```bash
flutter build apk --release
# অথবা প্লে স্টোরের জন্য
flutter build appbundle --release
```
এরপর Google Play Console-এ (একবারের $25 রেজিস্ট্রেশন ফি) আপলোড করুন।

## সততা সংক্রান্ত গুরুত্বপূর্ণ নোট
এই অ্যাপে "সচরাচর আসা প্রশ্ন" ব্যাজ ব্যবহার করা হয়েছে পূর্বের পরীক্ষায় ঘন ঘন আসা প্রশ্ন বোঝাতে।
কোনো পরীক্ষায় ঠিক কত শতাংশ প্রশ্ন কমন পড়বে তা নিশ্চিতভাবে বলা সম্ভব নয় — তাই অ্যাপে এমন কোনো
নির্দিষ্ট শতাংশ (যেমন "৯৬% কমন") দাবি না করাই ভালো, এতে ইউজারের আস্থা ও Play Store-এর নীতিমালা
উভয়ই রক্ষা পাবে। এর বদলে "নিয়মিত আপডেট হওয়া প্রশ্নব্যাংক" এবং "পূর্ববর্তী পরীক্ষার প্যাটার্ন
অনুসরণ করে সাজানো" — এই ধরনের সত্য ও যাচাইযোগ্য বার্তা ব্যবহার করা হয়েছে।

- "ভুল প্রশ্ন রিভিউ" ফিচার (Result স্ক্রিন থেকে ভুল উত্তর দেওয়া প্রশ্নগুলো আলাদা লিস্টে দেখানো)
